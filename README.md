📌 Overview
GuardianX is a next‑gen cybersecurity solution designed to protect digital identities. It detects ransomware activity and alerts users about personal data leaks in real time using free APIs and a secure GUI.
🚀 Features
- Ransomware Detection – Monitors suspicious file activity and encryption patterns.
- Data Leak Alerts – Integrates with APIs (e.g., HaveIBeenPwned) to check if user credentials are exposed.
- User‑Friendly GUI – Clean dashboard with alerts, breach details, and severity indicators.
- Recruiter‑Friendly Branding – Professional design with GuardianX theme and tagline.

📖 How to Run
- Clone the repo:
git clone https://github.com/DarkVibeCoders/GuardianX.git
cd GuardianX
- Install dependencies:
pip install -r requirements.txt
- Run the GUI:
python gui.py
- Enter email/username → get breach results + ransomware alerts.
